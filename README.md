# Anomaly Detection Dashboard

A React TypeScript frontend for a neural-based anomaly detection system. Displays log analysis results, anomaly alerts, model performance metrics, and real-time monitoring.

## Features

- Real-time log monitoring
- Anomaly alerts and notifications
- Interactive data visualizations (charts, graphs)
- Log data table with pagination, filtering, and search
- Model performance metrics display
- User authentication (login/logout)
- Responsive design (mobile, tablet, desktop)
- Dark/Light theme toggle
- Export functionality (CSV, PDF)
- Settings and configuration pages

## Tech Stack

- React 18 + TypeScript
- Vite
- React Router
- Recharts
- jsPDF
- date-fns

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173). Use any email/password to sign in (demo mode).

## Build

```bash
npm run build
npm run preview
```
